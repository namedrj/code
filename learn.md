# 创建仓库
```
两种创建方式：
手动进入目录：git init
自动创建目录：git init (directory)
```
# 基本用法
```
$ git init code
$ cd code/
$ git status
On branch master
$ git add .
$ git commit -m '(notes)'
$ git checkout (branch)
```
